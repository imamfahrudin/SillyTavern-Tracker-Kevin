An extension for SillyTavern that provides a customizable tracking feature to monitor character interactions and story elements.

## Original Repository

This extension was originally created by [Kaldigo](https://github.com/kaldigo).  
You can find the original repository here: [https://github.com/kaldigo/SillyTavern-Tracker](https://github.com/kaldigo/SillyTavern-Tracker)

## Installation

Install through SillyTavern's extension manager: Go to **SillyTavern > Manage Extensions > Install Extensions** and select this extension.

## Features

- Customizable tracking for character interactions
- Monitor story elements
- User-friendly interface
